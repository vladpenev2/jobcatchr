# Plan 08: User Settings Page

<objective>
Build the settings page where users can view/edit their profile, re-sync their LinkedIn data, and change their password.
</objective>

<tasks>

<task id="01-08-01">
<title>Settings page</title>
<details>
Create `src/app/(app)/settings/page.tsx`:

1. Fetch current user profile from `profiles` table
2. Display form with:
   - Name (editable)
   - Location (editable)
   - LinkedIn URL (read-only, set by admin)
   - Last LinkedIn sync date
3. "Save Changes" button: updates name + location via Supabase update on profiles
4. "Re-sync LinkedIn Profile" button: calls POST /api/admin/users/[id]/sync-profile (reuse the same endpoint, but allow users to sync their own profile)
5. "Change Password" section: current password not needed (Supabase handles it), just new password + confirm. Use `supabase.auth.updateUser({ password })`.

Use shadcn Card, Input, Label, Button, Separator components. Simple form with React state.
</details>
