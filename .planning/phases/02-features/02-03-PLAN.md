<task id="02-03-03">
<title>Find People API route</title>
<details>
Create `src/app/api/jobs/[id]/find-people/route.ts`:

POST: Run insider connections search
- Get the job from jobs table
- Get the user's profile_data (experiences, education)
- Check people_searches cache for this user+job combo
- If cached: return cached results
- If not:
  1. Auto-generate query: "[job title] or hiring manager or recruiter at [company] in [user location]"
  2. Accept optional custom query override from request body
  3. Resolve target company to LinkedIn numeric ID (via company.ts)
  4. Resolve user's past companies to numeric IDs (in parallel)
  5. Run Exa.ai people search
  6. Build LinkedIn search URLs
  7. Cache everything in people_searches table
  8. Return results

GET: Return cached results for this user+job (if any)
</details>
